# NestJS Hello World Scaffold – Backend Specification

## Project Title  
NestJS Hello World Backend Scaffold  

## Overview  
A minimal NestJS backend scaffold that runs a basic server and returns a “Hello World” response.

## Scope  
This project includes only:  
- Basic NestJS setup  
- Default `AppModule`, `AppController`, and `AppService`  
- A running backend server  

## Technology Stack  
- NestJS  
- Node.js  
- TypeScript
